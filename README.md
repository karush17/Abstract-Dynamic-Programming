## Abstract Dynamic Programming

This repository contains all the self-study material required for [Abstract Dynamic Programming](http://web.mit.edu/dimitrib/www/Abstract_DP_2ND_EDITION_Complete.pdf) (2nd Edition) by Dimitri Bertsekas. The contents of this repository provide the original chapters, self-contained notes and my solutions to exercises. 

* [`chapters`](chapters)- Contains original chapters from the monograph and the complete book.
* [`notes`](notes)- Contains self-study notes.
* [`solutions`](solutions)- Contains my solutions to exercises.
